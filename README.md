# Operation Management

## Overview
このリポジトリは、iPhone売上データの分析と需要予測を行うプロジェクトです。

## プロジェクト内容

### iPhone 16 売上予測分析
iPhone 12〜15の実績データを基に、iPhone 16の売上予測を実施しました。

#### 予測結果サマリー
- **予測期間**: 2024年9月〜2025年8月（発売後12ヶ月）
- **初年度予測売上**: 161.03百万台
- **予測手法**: iPhone 15をベースラインとして3%成長率を適用
- **過去実績比較**:
  - iPhone 12: 189.09百万台
  - iPhone 13: 149.11百万台 (-21.14%)
  - iPhone 14: 148.66百万台 (-0.31%)
  - iPhone 15: 156.33百万台 (+5.16%)
  - iPhone 16 (予測): 161.03百万台 (+3.00%)

## ファイル構成

### データファイル
- `Iphone_Sales_Data(1).csv` - iPhone 12〜15の日次売上実績データ（2020年10月〜2024年9月）

### 分析スクリプト
- `iphone16_forecast.py` - 需要予測モデルの実装
  - 月次集計
  - 時系列分析
  - 成長率計算
  - Excel出力機能

### 出力ファイル
- `iPhone16_Sales_Forecast.xlsx` - 予測結果のExcelファイル
  - シート1: iPhone16_Forecast - 月別予測データ
  - シート2: Historical_Comparison - iPhone 12〜16の並列比較
  - シート3: Summary - 主要統計とモデル手法
  - シート4: Historical_Monthly_Data - 元の月次実績データ

## 使用方法

### 必要なライブラリ
```bash
pip install pandas numpy openpyxl
```

### 予測スクリプトの実行
```bash
python3 iphone16_forecast.py
```

実行すると、`iPhone16_Sales_Forecast.xlsx`が生成されます。

## 予測モデルの特徴

1. **履歴データ分析**: iPhone 12〜15の発売後12ヶ月間の売上パターンを分析
2. **季節性考慮**: ローンチ月からの経過月数に基づく売上変動を考慮
3. **成長率適用**: 直近モデル（iPhone 15）をベースに保守的な成長率（3%）を適用
4. **モデル比較**: 過去4世代の売上推移から市場トレンドを把握
